<h1>CRUD Produits – PHP & MySQL</h1>

<h2>Description</h2>
<p>
Application web CRUD simple développée en <strong>PHP</strong> et <strong>MySQL</strong>
(niveau étudiant).
Elle permet d’ajouter, modifier, supprimer et afficher des produits.
</p>

<h2>Fonctionnalités</h2>
<ul>
  <li>Liste des produits (READ)</li>
  <li>Ajouter un produit (CREATE)</li>
  <li>Modifier un produit (UPDATE)</li>
  <li>Supprimer un produit (DELETE)</li>
</ul>

<h2>Technologies</h2>
<ul>
  <li>PHP</li>
  <li>MySQL (phpMyAdmin)</li>
  <li>XAMPP</li>
</ul>

<h2>Installation (local)</h2>
<ol>
  <li>
    Copier le dossier du projet dans <code>htdocs</code> :
    <br>
    <code>/Applications/XAMPP/htdocs/CRUD-PHP-MySQL</code>
  </li>
  <li>Démarrer <strong>Apache</strong> et <strong>MySQL</strong> dans XAMPP</li>
  <li>Importer le script <code>sql/script.sql</code> dans phpMyAdmin</li>
  <li>
    Ouvrir dans le navigateur :
    <br>
    <code>http://localhost/CRUD-PHP-MySQL</code>
  </li>
</ol>
